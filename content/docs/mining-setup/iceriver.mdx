## IceRiver KAS Setup Guide

This guide covers setup for IceRiver KAS-X1 and KAS-X2 ASIC miners.

### Quick Configuration

```bash
Pool URL: stratum+tcp://pool.katpool.io:15000
Worker: YOUR_KASPA_WALLET_ADDRESS
Password: x
```

### Web Interface Setup

1. Connect to your miner's web interface (default: 192.168.1.254)
2. Login with default credentials (admin/admin)
3. Navigate to "Miner Configuration"
4. Enter the pool details:
   - URL: `stratum+tcp://pool.katpool.io:15000`
   - Worker: Your Kaspa wallet address
   - Password: x

<Banner type="warning">
Remember to change your default admin password for security!
</Banner>
