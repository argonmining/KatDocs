## Bitmain K1/K2 Setup Guide

Configuration guide for Bitmain K1 and K2 ASIC miners.

### Quick Configuration

```bash
Pool URL: stratum+tcp://pool.katpool.io:15000
Worker: YOUR_KASPA_WALLET_ADDRESS.WORKER_NAME
Password: x
```

### Web Interface Setup

1. Access your miner's web interface
2. Navigate to "Miner Configuration"
3. Configure pool settings:
   - URL: `stratum+tcp://pool.katpool.io:15000`
   - Worker: Your Kaspa wallet address
   - Password: x

<Banner type="important">
The WORKER_NAME is optional but helps identify different miners in your dashboard.
</Banner>
