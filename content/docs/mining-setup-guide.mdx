## Quick Start Guide

This guide will walk you through setting up your ASIC miner with Kat Pool. We currently support IceRiver and Bitmain Kaspa ASICs, with Goldshell support coming soon.

## Supported Hardware

* IceRiver KAS-X1
* IceRiver KAS-X2
* Bitmain K1
* Bitmain K2
* Coming Soon: Goldshell KAS-1/KAS-2

## Basic Requirements

* Supported ASIC miner
* Kaspa wallet address
* Stable internet connection
* Proper cooling setup
* Adequate power supply

## Miner Configuration

### IceRiver Setup
```bash
Pool URL: stratum+tcp://pool.katpool.io:15000
Worker: YOUR_KASPA_WALLET_ADDRESS
Password: x
```

### Bitmain Setup
```bash
Pool URL: stratum+tcp://pool.katpool.io:15000
Worker: YOUR_KASPA_WALLET_ADDRESS
Password: x
```


<Banner type="important">
Replace YOUR_KASPA_WALLET_ADDRESS with your actual Kaspa wallet address. The WORKER_NAME is optional and can help identify different miners.
</Banner>
